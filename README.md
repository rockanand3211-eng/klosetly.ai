# STYLE.AI — Fashion Intelligence Dashboard

A modern Fashion AI dashboard built with Vite, React, TypeScript, and Tailwind CSS v4.

## Features

- Dark-themed UI with lavender accents matching the STYLE.AI design
- Sidebar navigation (Home, Wardrobe, Style Analysis)
- Photo upload section with AI Analysis CTA
- Product recommendations grid with oval cards, match badges, and filters

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start development server |
| `npm run build` | Production build         |
| `npm run preview` | Preview production build |

## Stack

- [Vite](https://vite.dev/)
- [React 19](https://react.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (icons)
